Fossils of Parks Township - Catalog

An interactive specimen catalog. It uses several things to speed up loading and consumption. Lazy-loaded photos, browser history objects, and local data storage all help achieve quick initial and return visit loading.

  • Interactive search filtering. Parses data in the browser.
  • Return links for searches, modal pages, and specimen pages.
  • Local storage with versioning for the data set. There's no need to fetch it again if it's not updated.
  • Custom PHP thumbnailing creates optimized thumbnails and large-sized specimen photos
